## Project Overview

This project was developed as part of the Software Development Engineer (SDE) – Full Stack Technical Assignment.

The goal was to accurately translate the provided Figma design into a production-ready frontend application while maintaining:

- Pixel-perfect UI implementation
- Responsive design across Desktop, Tablet, and Mobile
- Reusable component architecture
- Clean and maintainable code structure
- Smooth user interactions and transitions
- Performance optimization

## Setup Instructions

### Prerequisites

Make sure the following are installed:

- Node.js (v18 or above)
- npm
- Git

Verify installation:

```bash
node -v
npm -v
git --version
```

---

### Clone the Repository

```bash
git clone ### Clone the Repository

```bash
git clone https://github.com/godwinm8/N7-Modern-Banking.git
```

---

### Navigate to the Project Folder

```bash
cd N7-Modern-Banking
```

---

### Install Dependencies

```bash
npm install
```

---

### Run the Development Server

```bash
npm run dev
```

The application will start locally at:

```text
http://localhost:5175
```

---

### Create Production Build

```bash
npm run build
```

---

### Preview Production Build

```bash
npm run preview
```

## Development Approach

1. Converted Figma layouts into reusable React components.
2. Implemented CSS Modules for isolated styling.
3. Built responsive layouts using Flexbox and media queries.
4. Added smooth transitions and hover effects.
5. Optimized responsiveness for mobile, tablet, and desktop screens.
6. Ensured visual consistency with the original design.
